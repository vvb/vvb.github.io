Tracking Books Read in 2019

03 January 2019
  1. Catcher in the rye by J.D. Salinger
  2. Deep Work: Rules for Focused Success in a Distracted World by Cal Newport - Notes
  3. Peak: Secrets from the New Science of Expertise by K. Anders Ericsson and Robert Pool

Notes:
Emphasizes on "Deliberate practice modelled by following the best in the area". It provides a lot of examples, some in painfully lot of detail. It becomes too repetitive and boring by the end. It is in some ways similar to the book "Outliers" and can be skipped if a similar book has been read before.
